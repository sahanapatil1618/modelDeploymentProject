# 🚀 Model Deployment Project

## 📌 Overview
This project demonstrates deployment of a deep learning / machine learning model using **FastAPI** and **Docker**.  
It exposes a REST API endpoint that accepts image input and returns model predictions.

---

## ✨ Features
- REST API using FastAPI
- Model inference endpoint (`/predict`)
- File upload support (image input)
- Swagger UI documentation (`/docs`)
- Docker containerization for easy deployment

---

## 🛠 Tech Stack
- Python
- FastAPI
- Scikit-learn / PyTorch (based on model used)
- Uvicorn
- Docker
